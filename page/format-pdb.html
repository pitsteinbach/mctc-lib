Extensions

No extension implemented to the original format.

Missing Features

The following features are currently not supported:

  • Support for multiple file PDB input is not available
  • Fractional side occupation is currently not supported all optional sides count as full atoms
  • Cell information is not preserved, PDB input is always handled molecular

Note

Feel free to contribute support for missing features or bring missing features to our attention by opening an issue.
